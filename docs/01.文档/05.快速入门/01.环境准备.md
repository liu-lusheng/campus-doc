确保安装并正确配置一下软件：

## 后端环境

- JDK1.8

- Maven

  需替换成阿里云镜像加速

  ```xml
  <mirrors>
      <mirror>
          <id>alimaven</id>
          <name>aliyun maven</name>
          <url>http://maven.aliyun.com/nexus/content/groups/public/</url>
          <mirrorOf>central</mirrorOf>
      </mirror>
  </mirrors>
  ```

  

- mysql 5.7

- Redis

  win版本：<https://github.com/microsoftarchive/redis/tags>

  运行文件夹下面的`redis-server.exe`即可启动

- IDEA需安装LOMBOOK插件

## 前端环境

- Node Js 16或以上版本。
