## jar包方式

### 打包

项目根目录使用如下maven命令打包：

```sh
mvn clean package
```

如果使用的是IDEA工具，右上角的 `Maven -> Lifecycle` 中，依次点击 `clean` `package`

之后在  target 目录，可看到打好的 `coze-discord-xxx.jar` 包

### 部署

需安装好需要的环境

在jar包目录下，启动命令：

```sh
nohup java -jar xxxxx.jar > log.log &
```

启动之后在当前目录的 `log.log` 是启动日志
